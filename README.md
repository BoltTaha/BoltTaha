## 💡 **Featured Projects**

### 🧠 [**Rabt — Graph-RAG Code Intelligence**](https://github.com/BoltTaha/Rabt)
*Python • NetworkX • Gemini AI*

Hybrid AST + runtime instrumentation for intelligent code analysis. Achieved **99.2% context reduction** on real codebases — extract minimal subgraphs for LLM queries.

### 🏀 [**Basketball Made-Basket Detector**](https://github.com/BoltTaha)
*Python • YOLOv8 • OpenCV | Freelance Project*

Real-time video analytics pipeline fine-tuned YOLO model with state-machine event logic. Auto-exports 20-second highlights with timestamped accuracy.

### 💳 [**QR Payment Verification System**](https://github.com/BoltTaha/qr-payment-verification-system)
*JavaScript • Express • MongoDB • React • Gemini Vision*

Production fraud detection supporting 8+ Pakistani payment apps. Async pipelines, Gemini OCR, perceptual hashing, risk scoring. **Live on AWS EC2**.

### 📝 [**SnapTeX — Handwritten to LaTeX**](https://github.com/BoltTaha/SnapTeX)
*Python • Streamlit • Gemini AI*

Convert handwritten notes (PDFs/images) to clean LaTeX. Parallel batch processing, diagram recognition (UML/flowcharts), full session isolation.

### 🗜️ [**Context Window Compressor**](https://github.com/BoltTaha/Context-Window-Compressor)
*Python • Gemini • Gradio*

3-tier LLM memory system compressing conversation history into summaries + extracted facts. Rate limiting, exponential backoff, live UI.

### 🎯 [**Crisis Intelligence Decision Support**](https://github.com/BoltTaha/crisis-intelligence-decision-support)
*Python • Flask • scikit-learn • TensorFlow*

Full-stack ML system analyzing 105K+ articles. Logistic Regression + Neural Networks, 5-class categorization, A* search, Flask dashboard with explainability.
